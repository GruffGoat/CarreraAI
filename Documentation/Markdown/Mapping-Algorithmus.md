# Mapping Algorithmus

Dieser wurde Aufgrund der Tatsache, dass beliebig verschiedene Kurvenkonstellationen (theoretisch) auftreten 
können, verworfen. Dieser Algorithmus geht davon aus, dass die Kurvenstücke alle gleich lang und denselben Radius 
aufweisen. Grundsätzlich wurde dieser dazu entworfen, um herauszufinden, ob man sich auf der inneren oder Äußeren 
Spur befindet.
