# Neural Network Example Concept
Das angeführte Neuronale Netzwerk besteht aus einem Input Layer 2 Hidden Layer und einem Output-Layer auf das Ooutput Layer folgt ein Comparator der die 
beste operation auswählt und mit dieser dem PWM Generator steuert. Die Inputs sind momentan noch nicht ausreichend definiert sowie die Verknüpfung der einzelnen Neueronen.
